# **PY-EXEConvertor**
This program enables you to convert Python files to .EXE format more easily and conveniently via a single GUI, rather than dealing with PyInstaller one by one through the command prompt.

## **Points to consider**
♣ First, the Folder Containing the Python File Must be Selected.

♣ The ICO File Must be Inside the Selected Folder.

♣ The Code in The TextBox Can be Executed After the Target File is Opened via cmd.

♣ If the **.EXE** file fails to open due to an **icon file error**, move the **.EXE file out** of the **"dist"** folder and into the folder containing the problematic icon file.
